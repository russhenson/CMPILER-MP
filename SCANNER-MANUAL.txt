INSTRUCTIONS ON HOW TO USE THE SCANNER:
1. To put a source code in the textarea, you have two options:
    - First, by opening a file. Click the open found in file tab then select your file/test case
    - Second, by entering/pasting your source code in the textarea directly
2. To run the code
    - Click the run button
3. To test another code
    - Click new option found in file tab. The new option will reset the outputfile.tok and erase the previous output
4. To close the scanner
    - Click the new option again. This will reset everything and avoid having to delete the previous output manually
    - Close the software
